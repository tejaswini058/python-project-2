# 📁 Folder Manager

A command-line tool built in Python to manage your folders 
and files directly from the terminal — no GUI needed!

## What it does
- View all files and folders inside a directory
- Count total files and folders
- Search, rename, copy, and delete items
- Create new files and folders
- Check file size and last modified time
- Auto-arrange files into a single folder
- Remove empty folders
- Generate a text report of folder contents

## Technologies Used
- Python
- os module (built-in)
- shutil module (built-in)
- time module (built-in)

## How to Run
1. Make sure Python is installed on your system
2. Download or clone this repository
3. Run the following command:
   python folder_manager.py
4. Enter your folder path when prompted
5. Choose options from the menu (1-15)
